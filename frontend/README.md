# Frontend - Stock Prediction App

React + Vite frontend for the Stock Prediction application.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Features

- Modern React with Vite for fast development
- Interactive stock prediction form
- Real-time chart visualization with Chart.js
- Responsive design
- Error handling and loading states
- Clean, professional UI

## API Integration

The frontend communicates with the Flask backend at `http://localhost:5000/api/predict`

Make sure the backend is running before starting the frontend.
